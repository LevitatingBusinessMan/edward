= Tips for Edward

== File modification date in AsciiDoc
In standalone mode AsciiDoc will create a footer with the current time.
Because Edward recreates the file during each build, the timestamp is updated each time.
You may use ERB to set the doctitle to the files modification date instead.

[,adoc]
----
:docdatetime: <%= File.mtime(@page.path) %>
----

== Deploying websites

I personally like deploying through `rsync`:

[,console]
----
$ rsync -aiv --delete _site/ some_remote_server:/srv/www/example.com
----

== Default template options

To change the default template options just monkey patch the template's prepend method in your `_setup.rb`.

[,ruby]
----
# make asciidoctor produce standalone output per default
class Tilt::AsciidoctorTemplate
  def prepare
    @options[:standalone] = true if @options[:standalone].nil?
    super
  end
end
----
